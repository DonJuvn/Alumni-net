Басты бет Жаңалықтар Хабарландыру Достар

Бүгінгі қонағымыз Талғат Пардабеков,- JIHC колледж 2009 жылғы түлегі,- АҚШ, Вест Лафает қаласы Пурдю университетінің 2015 жылғы түлегі,- қазіргі таңда халықаралық Epam компаниясының бас Java программисті. Кездесуге, IT 3 курс, Мобильді қосымша бағытынан оқу практикасын өтіп жатқан студенттермен болды. Талғат аға өзінің үлкен тәжірибесімен бөлісіп және IT дың Java бағыты, алғашқы стажировка, шетелде оқу және де тағы басқа өзекті мәселелер аясында болды.
